## ソフトウェアII 第6回課題

## 実施した内容

基本課題である課題1（`huffman0`）に取り組んだ。発展課題には取り組んでいない。

- Ubuntuにおける`tree`と同様の方法で、ツリー構造を表現した。
  - 罫線素片のユニコード文字を利用している。
- 末端のノードでは、そのシンボル、コード語、出現回数を表示するようにした。
  - コード語は、左のノードに0、右のノードに1を割り当てている。
- 末端以外のノードでは、そのノードより下のシンボルの出現回数の合計を表示するようにした。
- 改行は`\n`として表示している。
